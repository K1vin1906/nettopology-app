NetTopology

Network Topology Visualizer for iOS

NetTopology is a network diagnostics and topology visualization tool for iPhone. Discover devices on your local network, run Ping and Traceroute, inspect Wi-Fi gateway and DNS, and understand your network at a glance — all with a Liquid Glass interface designed for iOS 26.

Features

  • Real-time external IP, ISP, and approximate location.
  • Ping and Traceroute via unprivileged ICMP datagram sockets.
  • Local device discovery via mDNS/Bonjour and HTTP banner probing.
  • Wi-Fi gateway and DNS resolver detection.
  • Force-directed network topology graph (coming soon).

Privacy

NetTopology does not collect personal information, has no backend server, and uses no third-party analytics SDKs. Read the full Privacy Policy.
